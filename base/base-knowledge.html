# 数据类型

基本数据类型 基本数据类型默认值 大小/位 对应包装类型 可表示数据范围
byte 0 8 Byte -128~127
short 0 16 Short -32768~32767
int 0 32 Integer -2147483648~2147483647
long 0L 64 Long -9223372036854775808~9223372036854775807
float 0.0f 32 Float 1.4E-45~3.4028235E38
double 0.0d 64 Double 4.9E-324~1.7976931348623157E308
boolean false 1 Bollean 0~65535
char 空 16 Character true或false

# 缓存池

  • new Integer(123) 每次都会新建一个对象
  • Integer.valueOf(123) 会使用缓存池中的对象，多次调用会取得同一个对象的引用。
